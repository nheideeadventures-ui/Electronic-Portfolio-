Lesson 1

Purpose: To create dynamic, interactive user experiences, handling everything from UI rendering (like React for Facebook), content loading, real-time updates, personalized feeds, video controls, and tracking user activity (Meta Pixel).

Static vs Dynamic Websites

Static Website: Delivered exactly as stored. Example: Photographer’s portfolio

Dynamic Website: Displays different content and interacts with users. Example: Social Media Platforms

JavaScript Overview

  • Most popular scripting language on the Internet
  • Recognized by all major browsers
  • Developed by Brendan Eich of Netscape
  • Enhances webpages and can build games
  • Licensed for organizations like Mozilla Foundation

Features & Examples

  • Google Search – real-time updates
  • YouTube – dynamic content & drag-and-drop playlists
  • Social Media – infinite scrolling & interactive menus
  • React (Facebook) – JS framework for dynamic UI

Lesson 2

JavaScript can be executed depending on where it's placed in HTML: HEAD, BODY, or external file.

Head

<head>
  <script type="text/javascript">
    function intro() {
      alert("Welcome Visitor!");
    }
  </script>
</head>
<body onload="intro()"></body>

Body

<body>
  <script type="text/javascript">
    document.write("Message by JavaScript");
  </script>
</body>

External JS

<script type="text/javascript" src="myJavaScript.js"></script>

Lesson 3

  • Interpreted Language: Requires a browser to run; source code can be viewed.
  • Scripting Language: Performs tasks by giving instructions to programs (like HTML).
  • Client-Side: Runs on the user’s computer.
  • Object-Oriented Programming: Treats elements as objects with methods (functions) and properties.

Key Terms

  • Object: Items in the browser (window, document, date)
  • Method: Functions performed on objects
  • Property: Attributes of objects
  • Event: User actions (click, hover, load)
  • Parameter: Data describing objects
  • Type: Variable type (integer, string, boolean)
  • Type Conversion: Converting a variable from one type to another

Common Objects & Methods

  • Window: location, open, setTimeout
  • Document: write, fgColor, cookie, lastModified
  • Math: PI, pow(a,b), max(a,b), min(a,b)
  • Navigator: appName, appVersion, cookieEnabled, platform

Lesson 4

Events

Actions triggered by mouse clicks, keypress, page load, etc.

  • onload – when page finishes loading
  • onunload – when user exits page
  • onclick – triggers a function
  • onmouseover / onmouseout – mouse hover effects
  • onsubmit – validates form before submission

Popup Boxes

  • Alert: alert("message");
  • Prompt: prompt("Enter name","default");
  • Confirm: confirm("Are you sure?");

Variables

Temporary storage for values and expressions. Rules:

  • Case-sensitive
  • Must start with letter or underscore
  • No spaces or reserved words
  • Can hold numbers, strings, booleans, arrays, objects
var x = 5;
var y = x + 5;
var fan = "Standard";
var isDone = false;

Operators

Arithmetic: +, -, *, /, %, ++, --

Comparison: ==, ===, !=, !==, >, <, >=, <=

Logical: && (AND), || (OR), ! (NOT)

Conditional: variablename = (condition) ? value1 : value2;
